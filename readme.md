# entity-space
entity-space is a data layer utility for TypeScript web applications.

it helps you with:
* caching data fetched from APIs
* unifying data loading logic
* abstracting multiple APIs into a single interface


introducing it into your project requires:
1) describing properties and relations of your entities
2) telling entity-space how to load, save and delete them
3) using the in-built query system to execute those operations
